Robot Operating System

The robot operating system ROS has emerged as the most popular middleware for complex robots

Advantages: parameters, logging, visualization, SLAM, moveit, ...

Disadvantages: difficult to setup, regression during migration to ROS 2, lack of package updates

Also: Lack of security, structural problems with Gazebo simulator

Lack of Resources

Maintaining a large open-source software project is hard work

Most of these problems are solvable

Lack of developers

Lack of money

Example: OpenCV

Most popular library for computer vision

Integral part of ROS

Planned major upgrade for version 5. Crowdfunding campaign floundering (10,000 USD of $50,000 USD)

Specification

Distributed processing system to isolate behaviors (Subsumption architecture, behavior trees)

Nodes communicate data via publish and subscriptions (pubsub)

Early World-wide Web

The web is based on open standards

Internet started based on Hypertext Markup Language (HTML) to describe content

Cascading Style Sheets (CSS) to define formatting

Javascript to define behavior

Paradigm: powerful servers push data and some light processing to clients/browsers

Modern Web Stack (cont.)

Web is a distributed platform from the start

Representational State Transfer (REST) provides a architecture to orchestrate services (uniform interfaces, cacheable, client-server, stateless, layered system, code on demand)

Security is a primary focus of web computation

Modern Web Stack

Access to hardware in the client: camera, microphone, bluetooth, ...

Webassembly is a stack based virtual machine for the javascript runtine. Compile many languages to run in the browser (C++, python, rust, ...). At the moment lacks memory management, which will be included in the next standard

Modern Web Stack (cont.)

WebGL provides GPU accelerated graphics

WebGPU provides advanced graphics and GPU-based computation

Websockets allow app to create network connections

WebRTC provides real-time communication between clients

Modern Web Stack (cont.)

Modern browser has developed into a universal operating system

Runs on anything that supports a modern browser: windows, mac os, linux, android, iOs, switch, ...

Performs computation at almost native speed

Client can do heavy lifting in computation

Resources

Enourmous economic engine

Thousands of developers at Google, Microsoft, Facebook work on improved javascript engine and other web technologies

Lots of (open-source) projects to create better applications

Issues and Concerns

Companies are trying to influence the web standards (e.g., Google DRM proposal)

A new framework every week: cgi, jquery, ..., angular, react, vue, svelte, nextjs, ...
